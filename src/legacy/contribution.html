У нас общая история!

давняя суджа 22 февраля 2021

Наш проект открыт для всех заинтересованных пользователей, которые могут участвовать в его развитии.

Вы можете:
  • предложить новые объекты для размещения QR-кодов и оказать содействие в их размещении;
  • разместить публикацию на нашем сайте;
  • поделиться интересной информацией.
Как связаться с нами? Информация на странице О нас.

История Суджанского края достойна того, чтобы о ней знало как можно больше людей. Поэтому наш проект активно развивается, мы рады сотрудничеству. Будем благодарны тем, кто перенесет наши статьи в Википедию или на другие вики-ресурсы.

Сайт находится в разработке, и мы приглашаем к сотрудничеству веб-разработчиков. Мы используем следующие технологии:

  • HTML5, SCSS + Bootstrap 4
  • JS + JQuery, в перспективе - Vue
  • Gulp + Webpack
  • Python 3.9 + Django (в перспективе)
Сейчас ведется активная работа по улучшению клиентской части нашего сайта и созданию функционала на бэкенде. Если вам интересно поработать над этим с нами, то более подробную информацию о взаимодействии можно посмотреть в readme проекта на Github.

Если вы нашли ошибку в коде или придумали, как его улучшить, напишите issue.

Наш телеграм-бот создается на Python 3.9, вы тоже можете принять участие в его разработке. Более подробно - в readme проекта на Github.

Давайте делать историю вместе!
